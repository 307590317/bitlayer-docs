# wBTC and Gas Fee

### wBTC

Bitlayer 使用 BTC 作为 Gas Token，一般情况下，都会有一个对应的 wBTC token 方便各类协议交互。

用户可以通过调用 Dex 的 swap 功能实现兑换。

#### Macaron Swap

用户在Macaron Swap界面把BTC兑换成wBTC

https://www.macaron.xyz/#/swapl
